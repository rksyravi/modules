
-- SUMMARY --

The "Site locaiton" module displays the site country, current time etc. to the site user.


-- INSTALLATION --

* Install as usual, see https://www.drupal.org/documentation/install/modules-themes/modules-8
  for further information.


-- CONFIGURATION --

* For admin configuration of site location. Use Url: {BASE_URL}/admin/config/user-interface/sitelocation.
* Block name: Site locaiton. Place this in the region wherever you want.
